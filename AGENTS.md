# Repository Instructions

## Python Environment
- Use the repository virtual environment at `.venv`.
- Run Python commands with `.venv/bin/python`.
- Run tests with `.venv/bin/pytest`.
- Run package management commands with `.venv/bin/pip` when needed.
- Do not assume `python`, `python3`, `pytest`, or globally installed packages point to the repo environment.

## Validation Commands
- Run `preflight.py` with `.venv/bin/python preflight.py`.
- Prefer the repo virtual environment for Streamlit and CLI entrypoints as well.
